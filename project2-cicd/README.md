# Project 2 - CI/CD Pipeline with GitHub Actions

## What this project does
Automatically runs tests every time code is pushed to GitHub.

## Tools Used
- GitHub Actions
- Python
- Pytest

## Pipeline Flow
Code Push → GitHub Actions Triggered → Tests Run → Pass/Fail Result

## Files
- app.py - Simple Python application
- test_app.py - Unit tests
- .github/workflows/cicd.yml - Pipeline configuration
